# 🎫 Support Ticket Classification & Prioritization (Task 2)

## 🚀 Project Overview
This project focuses on building a Machine Learning system to automatically classify customer support tickets and assign priority levels.

The system helps businesses reduce manual effort, prioritize urgent issues, and improve customer support efficiency.

---

## 🎯 Objective
- Classify support tickets into categories (e.g., Billing, Technical, Account)
- Predict ticket priority (High / Medium / Low)
- Improve response time and support operations using ML

---

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains customer support ticket details including:
- Ticket Subject
- Ticket Description
- Ticket Type (Category)
- Ticket Priority

---

## ⚙️ Steps Performed

### 1. Data Cleaning
- Converted text to lowercase
- Removed punctuation and special characters

### 2. Text Preprocessing
- Combined subject and description
- Cleaned raw ticket text

### 3. Feature Extraction
- Used TF-IDF vectorization to convert text into numerical features

### 4. Model Building
- Logistic Regression model for Ticket Type classification
- Logistic Regression model for Priority prediction

### 5. Model Evaluation
- Evaluated using classification report (precision, recall, F1-score)
- Used confusion matrix for performance analysis

---

## 📊 Results
- Successfully classified tickets into categories
- Predicted ticket priority effectively
- Model performs well for structured support data

---

## 💡 Business Impact
This system can:
- Automate ticket classification
- Reduce manual workload for support teams
- Identify high-priority issues quickly
- Improve response time and customer satisfaction

---

## ⚠️ Limitations
- Performance depends on data quality
- May struggle with unseen or ambiguous text
- Requires larger dataset for higher accuracy

---

## 🚀 Future Improvements
